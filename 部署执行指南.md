### 选项2：Vercel CLI（需要登录）

**步骤**：
1. 打开终端（Git Bash）
2. 执行以下命令：
```bash
cd "C:\Users\11943\WorkBuddy\2026-05-11-task-3\aiflow-mvp"
npx vercel login
# 按提示输入邮箱，点击邮件链接授权
npx vercel --prod
```
3. 按提示配置：
   - Set up and deploy? **Y**
   - Which scope? 选择你的账户
   - Link to existing project? **N**
   - What's your project name? **aiflow-mvp**
   - In which directory is your code located? **./dist**
   - Want to override settings? **N**

**优势**：自动HTTPS、自定义域名、Git集成

### 选项3：GitHub + Vercel（自动化部署）

**步骤**：
```bash

cd "C:\Users\11943\WorkBuddy\2026-05-11-task-3\aiflow-mvp"

git add .
git commit -m "Initial commit: AIFlow MVP"

# 2. 创建GitHub仓库（手动）
# 访问 https://github.com/new
# 仓库名：aiflow-mvp
# 不要初始化README

# 3. 推送代码
git remote add origin https://github.com/你的用户名/aiflow-mvp.git
git branch -M main
git push -u origin main

# 4. Vercel导入（手动）
# 访问 https://vercel.com
# Import Project -> 选择GitHub仓库
# 自动检测Vite配置 -> Deploy
```

**优势**：每次push自动部署、预览环境、团队协作
